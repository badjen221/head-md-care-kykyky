# 2025-05-20

Create a maquette in Unity of your cardboard prototypes using various "greyboxing" style techniques:

- Simple shapes (Hierarchy > New > Cube)
- Simple colors (Assets > New > Material)
- Place your camera
- Don't worry about code
- If simple cubes aren't good enough, you can use ProBuilder
    - [] (https://www.youtube.com/watch?v=VCt3UoGnlLI)


* Grey is a metaphor. Your scene does not have to be grey.
